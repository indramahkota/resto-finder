# Resto Finder
Example progressive web app using typescript, webpack, workbox, index db, and dicoding restaurant api.

From your command line, clone and run command bellow:

```bash
# Clone this repository
$ git clone https://github.com/indramahkota/resto-finder.git

# Go into the repository
$ cd resto-finder

# Install dependencies
$ npm install

#Start's development server
$ npm run start
```
